PROGRAM DESIGN AND DEVELOPMENT

PURPOSE :
Knowledge of problem solving and programming concepts are essential for those who develop applications for users. Hence, to provide the required knowledge,this course imparts basic knowledge in C Programming along with the concepts of design and development of programs using C.

Syllabus :

UNIT I : INTRODUCTION TO PROBLEM SOLVING AND PROGRAMMING
UNIT II : OVERVIEW OF C
UNIT III : FUNCTIONS
UNIT IV : ARRAYS, STRUCTURES AND UNIONS
UNIT V : POINTERS
